## Neovim Setup
Build my dream text editor :)

### Hot Keys

Key | Function 
--- | --------
jk  | esc
\>  | move block of text right
\<  | move block of text left
ctrl+p | open markdown preview
ctrl+n | open nerdtree (move cursor to nerdtree)
tt     | close nerdtree
ctrl+m | move cursor to editor
ctrl+\ | open/close ternimal
